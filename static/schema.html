This database contains the most popular baby names in NYC from 2011 to 2016

Field Type Description
year_of_birth year(4) The year of the child's birth.
gender varchar(6) The gender of the child.
ethnicity tinytext The mother's ethinic background.
first_name tinytext The child's first name.
count smallint(6) Total number of children with name
ranking tinyint(3) unsigned The rank of the given name
